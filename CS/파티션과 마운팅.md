# 파티션(Partition)

파티션은 **디스크와 같은 물리적 저장 장치를 OS가 독립적으로 관리할 수 있는 논리적 단위로 나누는 과정이다.**

하나의 디스크에 여러 개의 독립적인 파일 시스템을 사용하거나 여러 OS를 설치하기 위함이다.

일반적으로 데이터 영역 분리를 위해 파티션을 사용하며, 물리적인 하드디스크에 논리적인 파티션을 나누는 것을 말한다.

파티션을 나누고 제일 먼저 해야 하는 일은 파일 시스템을 지정해줘야 한다. 파일 시스템은 meta 영역과 data 영역으로 나누어져 있다. meta 영역에는 파일의 물리적 정보가 저장되고, data 영역은 실제 데이터가 저장되는 공간이다.

# 마운팅(Mounting)

마운팅은 OS가 외부 파일 시스템을 인식하고, 해당 파일 시스템을 기존 디렉터리 구조에 연결해 접근 가능하게 만드는 과정이다.
